# Platzi Play Java



Aplicacion en consola con Java [Curso Platzi](https://platzi.com/cursos/java/)

* Primero sae debe de compilar con el comando ``javac [fileName].java``. Con esto se crean los archivos ``.class`` que son los compilados.
* Para ejecutar se debe de usar el comando ``java [fileName].java``. Sin compilar, no se puede ejecutar por comandos.
